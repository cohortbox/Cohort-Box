📌 Overview

CohortBox is a full-stack web application that enables users to:

Communicate in group-based environments (cohorts)
Chat and interact in real-time
Manage user profiles and sessions
Experience a clean and responsive UI

Built with scalability in mind, this project reflects real-world architecture used in modern social platforms.

✨ Features
🔐 JWT Authentication (Login / Signup)
💬 Group chat system
👥 Cohort-based user organization
🔎 Search functionality
⚡ REST API with pagination
📱 Responsive UI
🧩 Modular backend structure
🛠️ Tech Stack
Layer	Technology
Frontend	React (CRA), CSS
Backend	Node.js, Express
Database	MongoDB (Mongoose)
Auth	JWT
